# Security Policy

This repository is an educational security lab. It contains controlled attack simulations to demonstrate why IoT access-control systems need layered defenses.

Do not use the simulation code against third-party systems, public networks, or devices you do not own or have explicit permission to test.

If you adapt this project for real-world use, replace the demonstration PIN masking with production-grade authenticated encryption, secure key storage, replay protection, and a hardened transport layer.
